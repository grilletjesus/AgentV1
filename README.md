## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/grilletjesus/agentv1.git
    ```
2. Navigate to the project directory:
    ```bash
    cd agentv1
    ```
3. Install dependencies using `uv`:
    ```bash
    uv install
    ```
